# DompixelBlog

Este é um desafio teste para vaga de front, projeto com Mantine e Tailwind CSS.

## Development

Rodar o projeto no localhost: (apenas isso mesmo, ao acessar a home, o posts estão inseridos no "firebase")

```bash
npm run dev
```

## More information

- [Mantine](https://mantine.dev/)
- [Tailwind CSS](https://tailwindcss.com/)
- [tailwind-preset-mantine](https://github.com/songkeys/tailwind-preset-mantine)
